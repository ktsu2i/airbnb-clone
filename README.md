# Airbnb Clone

## To visit the application, [click here](https://airbnb-clone-ktsu2i.vercel.app/).

## Purpose of This Project
- To have more practical experience in creating web applications with Next.js
- To understand how to implement:
  - specific components such as Calendar to book, Map to select a country
  - a functionality to filter items with query parameters

I gained the skills through [the Airbnb Clone tutorial video](https://www.youtube.com/watch?v=PGPGcKBpAk8). 
Thank you for the awesome tutorial, Antonio!

## Functionalities
- Sign-up, login, logout
- Book places and check them in My trips
- Click like buttons to save the liked places in My favorite
- Upload info to host your places through Airbnb your home
- Check your places in My properties
- See reservations of your places in My reservations
- Filter places by category, country, duration of stay, and a number of guests

### Additional Functionality
I added my own functionality to the app after I completed the tutorial.
It is working as desired but has one thing to fix (in progress).
- Change your icon image in Settings

## Technologies
- TypeScript
- React, Node.js, Next.js, NextAuth.js
- Tailwind CSS
- ORM (Prisma)
- MongoDB
- Deployed on Vercel
